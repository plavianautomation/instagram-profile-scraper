# Instagram Profile Scraper (Apify API)

Automation pipeline that extracts structured Instagram profile data for analytics and influencer research.

## Features
- Extracts profile metadata (followers, following, bio)
- Uses Apify Instagram scraping API
- Outputs structured JSON datasets

## Tech Stack
Python • API Integration • Apify • Data Pipelines

## Use Cases
- Influencer discovery
- Social media analytics
- Audience research

## Sample Output

[
  {
    "username": "natgeo",
    "followers": 282000000
  }
]

## Setup

pip install -r requirements.txt

Add your Apify token in scraper.py and run:

python scraper.py
